## 📌 Sobre o projeto

O **Turistou** é um aplicativo mobile desenvolvido em **Kotlin** para Android, criado como projeto acadêmico, com foco em turismo, localização e visualização de pontos de interesse por meio de mapas.

A proposta do app é oferecer uma experiência simples e intuitiva para usuários que desejam explorar locais turísticos, utilizando recursos de geolocalização e integração com mapas.

> Projeto acadêmico desenvolvido com foco em aprendizado de desenvolvimento mobile, boas práticas de organização e uso de recursos nativos do Android.

---

## ✨ Funcionalidades

- 🗺️ Visualização de mapa interativo
- 📍 Uso de recursos de localização
- 🧭 Apoio à exploração de pontos turísticos
- 📱 Interface mobile para Android
- 🔐 Configuração de chave da API do Google Maps via `local.properties`
